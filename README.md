🍕 **Tasty-Slice – Pizzeria Website**

Tasty-Slice is a responsive landing page developed for a fictional pizzeria. The site offers a complete user experience, allowing visitors to view the menu, explore the restaurant’s atmosphere, access contact information, and navigate to reservation and competition sections.

🚀 Technologies Used

- **HTML5:** Semantic structuring of the content.
- **Bootstrap 5:** CSS framework ensuring responsiveness and modern components (such as the Carousel and Navbar).
- **CSS3:** Custom styling through the file `assets/css/style.css`.
- **JavaScript:** Logic for automatic navbar collapse on mobile devices and smooth scrolling.
- **Font Awesome:** Vector icons for social media and menu details.
- **Google Fonts / Favicon.io:** Visual identity and browser tab icons.

📋 Features

- **Responsive Design:** Adapted for desktops, tablets, and smartphones.
- **Fixed Navbar:** Easy navigation with automatic closing on mobile after clicking.
- **Image Carousel:** Visual highlight of products in the header section.
- **Restaurant Gallery:** Display of the restaurant environment through image cards.
- **Dynamic Menu:** Listing of pizzas and drinks with prices and descriptions.
- **Social Media Links:** Integration with Facebook, X (Twitter), and Instagram in the footer.
- **Reservation System:** Call-to-action (CTA) button directing users to booking.html.
